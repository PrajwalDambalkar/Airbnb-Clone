Place your logo image here as `logo.png` so it will be served at `/photos/logo.png`.

Recommended filename: logo.png
Recommended size: width ~ 1400px (Vite will scale in the UI). The component uses a height of 34px for the header logo.

To use the attached image from the repo root, copy it into this folder:

macOS / Linux:
  cp /path/to/your/downloads/logo.png ./apps/frontend/public/photos/logo.png

Windows (PowerShell):
  Copy-Item C:\path\to\logo.png .\apps\frontend\public\photos\logo.png
